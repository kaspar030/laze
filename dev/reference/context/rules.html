laze Manual

rules

This fiels contains a list of laze build rules.

A laze build rule mostly correspondents to Ninja rules.

Example:

contexts:
  - name: default
    rules:
      - name: CC
        in: c
        out: o
        cmd: ${CC} ${in} -o ${out}

rule fields

  • name
  • description
  • cmd
  • in
  • out
  • options
  • gcc_deps
  • rspfile
  • rspfile_content
  • pool
  • always
