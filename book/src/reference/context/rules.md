# rules

This fiels contains a _list_ of laze build rules.

A laze build rule mostly correspondents to Ninja rules.

Example:

```yaml
contexts:
  - name: default
    rules:
      - name: CC
        in: c
        out: o
        cmd: ${CC} ${in} -o ${out}
```
